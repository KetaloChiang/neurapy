Neurapy

NeuraPy is a collection of python modules useful for analyzing data obtained in neurophysiological experiments.

View the Project on GitHub kghose/neurapy

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

Modules currently in NeuraPy are:

  • lablib - a module for loading lablib data files
  • monkeylogic - a module for loading data files saved by the MonkeyLogic program
  • cerebus - a module for loading .nev and .nsx files saved by BlackRock's Cerebus software
  • plexon - a module for loading .nex and .plx files produced by the Plexon Map system

See the Readme in the source for more details.
