NeuraPy is a collection of python modules useful for analyzing data obtained in neurophysiological experiments. Currently included are:
  • lablib.py - a module for loading lablib data files 
  • nev.py - a module for loading .nev files saved by BlackRock's Cerebus software 
  • nsx.py - a module for loading .nsx files saved by the Cerebus software
